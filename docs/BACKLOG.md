## 1. IA / вкладки (решения с обсуждения)

Текущие табы: Тренування · Програми · Аналітика · Здоров'я · Акаунт.

Решения пользователя:
- **Головна** — нужна, но над смыслом/наполнением думаем (today-хаб: снимок дня + старт + продовжити + рекорд).
- **Програми** — важная, оттуда стартуют все большие (предрасписанные) тренировки; будем много допиливать.
- **Тренування** (сейчас = история) — само по себе мало смысла. Нужна заметная кнопка **«Почати тренування» → выбор: з нуля / імпортувати програму**. История остаётся как перечень.
- **ІІ-коуч** — **отдельная вкладка** (нужна 100%). Чат + запись голосового (STT → текст). **Без TTS** — коуч не озвучивает (дорого по костам). Тексты гоняем через гейтвей/лимиты.
- **Аналітика** — оставляем, но **не по центру**; важный «залипательный» экран.
- **Здоров'я** — сейчас только под Машу/ауру. **Включать как вкладку условно**: только если юзер указал, что есть умный девайс, И подключил его.
- **Акаунт → Профіль** — переделать (сейчас «срала-мазала»). См. §6.

Открытый вопрос: при условном «Здоров'я» + новом «Коуч» получается 6 пунктов в нижнем меню. Варианты: 6 табов для Маши; либо снимок здоровья на Головній ведёт на полный экран без постоянного таба. Решить при вёрстке.

Ориентир по порядку: Головна · Програми · Коуч · Аналітика · Профіль (+ Здоров'я условно).

## 2. Программы и тренировки — управление данными
- [ ] **Редактировать программу после импорта** (переименовать, править блоки/упражнения/подходы, удалять упражнения).
- [ ] **Удалять программу.**
- [ ] **Удалять тренировку** (если случайно создал). Это **данные** → удаление **по подтверждению**: кнопка «Видалити» → **варнинг** «Цю тренування буде назавжди видалено з бази даних» → только потом удаляем. (двухшаговое, не моментальное.)

- [ ] **Голосовое сообщение**: запись аудио → STT (расшифровка в текст) → в чат. **TTS нет** (дорого) — ответ только текстом.
- [ ] Прогон через `ai-gateway` (роут `coach_chat` уже есть → Claude), учёт лимитов/бюджета.
- [ ] Контекст коуча: доступ к тренировкам/рекордам/готовності (на сервере, аккуратно по костам — не пихать всё в каждый запрос).
- [ ] Тон: поддерживающий, на языке пользователя (есть в ТЗ).
- Открытое: какой STT-провайдер и его косты (часть токеномики, §5).

## 5. Архитектура / косты / безопасность (отдельный цикл)
- [ ] **Пентест / секьюрка** отдельным проходом (`/security-review`): RLS, RPC, секреты, edge-функции, ключи.
- [ ] **Архитектурный ревизит**: для каждой кнопки понять, какой процесс тянет за собой другие (граф «действие → запросы/функции»), чтобы оптимизировать время/вычисления/**токеномику** (особенно ИИ).
- [ ] **Локально vs интернет**: разобрать, что работает офлайн, что требует сети; **минимизировать сеть** (не «заёбывать» юзера). Кандидат: офлайн-логирование тренировки с последующей синхронизацией.

## 6. Профіль (переделка) + онбординг
- [ ] **Пол** — выбор при онбординге И в Профілі: **ч / ж / інше (ввести своё) / не вказувати**.
- [ ] Если **ж** → предложить **відстеження циклу** (для аналитики по фазам).
- [ ] **«Мої дисципліни»** — сейчас «ни о чём» + термин «спеціальні словники» юзеру ничего не говорит. Договорились: упражнения из этих словарей **и так подтягиваются** (поиск пробивает всё), просто не перегружаем общий пул. → Переосмыслить экран: убрать жаргон, может скрыть концепт «словники», оставить лёгкое «що показувати у списку».
- [ ] **«Мої вправи»** — починить вёрстку (обрезает).
- [ ] **«Мої еспандери»** — оставить (ниша: сила хвата, пригодится мне и пацанам после релиза). Дать **кастомное название** (напр. «Хевик 300») даже **без RGC** (чел не мерял) — выбор из спарсенного списка ИЛИ своё имя.

## 7. «З кентом / тренером» (режим напарника)
- [ ] Слайдер/тогл в тренировке: отмечать не только свои, но и **результаты напарника** по тому же упражнению.
- [ ] В **личные рекорды/аналитику НЕ идёт**; удобно «в моменте».
- [ ] **Сохранять аккуратно** (кандидат: `sets.meta` с пометкой partner, или отдельная лёгкая таблица) — чтобы при желании потом достать.

## Порядок (черновой, на согласование)
1. **Данные-safety**: ред./удаление программ + удаление тренировки с варнингом + правка даты цикла. (быстро, чинит текущую боль)
2. **Профіль/онбординг**: пол → цикл, чистка дисциплін, фикс «Мої вправи», кастомный эспандер без RGC.
3. **ІІ-коуч** (большой модуль).
4. **IA-переезд** вкладок (Головна/Коуч/условное Здоров'я/порядок).
5. **Цикл архитектуры + пентест** отдельным циклом.
