# Пресет-аватарки

Картинки выбираемых аватарок. Бандлятся с апкой (не грузятся по сети).

## Как добавить аватарку
1. Положи файл сюда: `assets/avatars/<key>.png` (например `01.png`).
   - **Формат:** PNG (или JPG).
   - **Форма:** квадрат (отображается в круге).
   - **Размер:** ~256–512px по стороне (больше — лишний вес бандла).
   - **`<key>`** — короткий уникальный id (имя файла без расширения).
2. Впиши строку в реестр `src/lib/avatars.ts`:
   ```ts
   { key: '01', source: require('../../assets/avatars/01.png') },
   ```
   (RN требует статический `require()`, поэтому список явный — по строке на файл.)

После этого аватарка автоматически появится в пикере на экране аккаунта.
Выбор хранится в `profile.avatar` (ключ); если ключа нет — кружок с инициалами.
